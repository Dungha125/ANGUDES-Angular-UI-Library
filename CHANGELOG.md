# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-12-05

### Added
- Initial release of Angudes UI Component Library with 20+ components

#### Layout Components
- Box component with multiple variants and shadow options
- Card component with header, body, footer and loading state
- Card Header, Body, Footer sub-components

#### Navigation Components
- Navbar component with fixed positioning and light/dark variants
- Menu component with horizontal/vertical orientation and 3 variants (default, pills, tabs)
- Menu Item component with active state and router support
- Submenu component with expand/collapse functionality
- Tabs component with line, card variants
- Tab component for tab content
- Pagination component with size changer and quick jumper

#### Form Components
- Button component with 7 variants (primary, secondary, success, warning, danger, outline, ghost) and 3 sizes
- Input component with validation states, clear button, and form integration (ControlValueAccessor)

#### Data Display Components
- Carousel/Slideshow component with autoplay, navigation arrows, dots, and slide/fade effects
- Carousel Slide component
- Badge component with count, dot, and status variants
- Alert component with 4 types (success, info, warning, error)
- Progress component with status and active animation
- Spinner component with multiple sizes

#### Feedback Components
- Modal component with mask, keyboard support (ESC), and customizable footer
- Tooltip component with 4 placements and multiple triggers
- Dropdown component with click/hover triggers and 4 placements

#### Special Components
- Text Effect component with 6 animation effects (typing, fade, slide, glow, gradient, wave)
- Accordion component with accordion mode (single panel open)
- Accordion Panel component with expand/collapse

#### Features
- Comprehensive documentation in README.md
- Full TypeScript support
- SCSS styling with modern design inspired by Ant Design
- Form integration (ngModel, Reactive Forms)
- Router integration
- Animations & Transitions
- Accessibility support
- Responsive design
